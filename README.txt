# CUSCN30Python Project

Welcome to the CUSCN30Python project repository. This repository contains Python scripts and resources related to the CN dataset for ARCI and ARCIII, generated using ArcGIS with Python 2.7 and ArcGIS version 10.8.

## Project Contents

This project contains the following key files:

1. **TransfertoARCI.py**: This Python script is used to generate the CN dataset for ARCI based on the CONUS30 ARCII data for the same year.

2. **TransfertoARCIII.py**: This Python script is used to generate the CN dataset for ARCIII based on the CONUS30 ARCII data for the same year.

## System Requirements

To run the scripts, make sure you have the following:

- Python 2.7
- ArcGIS version 10.8

## Usage

1. Ensure you have the required Python and ArcGIS versions installed.

2. Open the respective Python script, i.e., "TransfertoARCI.py" or "TransfertoARCIII.py," using your Python environment.

3. Run the script to generate the CN dataset for ARCI or ARCIII.
